## Порядок выполнения работы

1. Операционная система Windows 10 установлена на виртуальной машине.

2. Вход в систему выполнен с использованием учетной записи, обладающей правами администратора.

3. Нажал сочетание клавиш `Win+R`, ввел `msconfig` и нажал ВВОД.

![Окно "Выполнить" с командой msconfig]

4. На вкладке **Общие** выбрал параметр **Выборочный запуск** и снял флажок в поле **Загружать элементы автозагрузки**.

![System Configuration - General tab]

5. На вкладке **Службы** установил флажок **Не отображать службы Майкрософт** и нажал кнопку **Отключить все**.

![System Configuration - Services tab]

6. Нажал кнопку **ОК**, затем нажал кнопку **Перезагрузка**.

7. Запустил командную строку от имени администратора (Пуск → правой кнопкой мыши на «Командная строка» → Запуск от имени администратора).

![Командная строка от имени администратора]

8. В командной строке ввел `sfc /scannow` и нажал Enter.

**Результат:**
```
Защита ресурсов Windows не обнаружила нарушений целостности.
```

9. Ввел команду `sfc /help` для выяснения параметров.

**Результат:**
| Параметр | Описание |
|----------|----------|
| `/scannow` | Сканирует целостность всех защищенных системных файлов |
| `/verifyonly` | Сканирует целостность без восстановления |
| `/scanfile=<путь>` | Сканирует указанный файл |
| `/verifyfile=<путь>` | Проверяет указанный файл без восстановления |
| `/offbootdir=<каталог>` | Каталог загрузки для автономного сканирования |
| `/offwindir=<каталог>` | Каталог Windows для автономного сканирования |

10. На диске C: создал папку `1isp3`. В папке создал текстовый документ `sfccommand.txt`.

11. Выполнил команду `sfc /verifyfile=C:\1isp3\sfccommand.txt`.

**Результат:**
```
Защита ресурсов Windows не может выполнить запрошенную операцию.
Указанный файл не является защищенным системным файлом Windows.
```

12. Выполнил команду `sfc /scannow` для проверки всех защищенных файлов.

**Результат:**
```
Защита ресурсов Windows не обнаружила нарушений целостности.
```

## Контрольные вопросы

**1. Почему команда Sfc не может проверить созданный Вами файл sfccommand.txt?**

Команда SFC проверяет только защищенные системные файлы Windows, имеющие цифровую подпись Microsoft. Созданный пользовательский файл не является системным и находится в пользовательской папке, поэтому SFC его не проверяет.

**2. С какими параметрами можно использовать команду sfc?**

`/scannow`, `/verifyonly`, `/scanfile`, `/verifyfile`, `/offbootdir`, `/offwindir`, `/offlogfile`, `/help`.

**3. В каких ситуациях нужно использовать «чистую загрузку»?**

Чистая загрузка используется для диагностики конфликтов программного обеспечения, устранения ошибок при запуске, поиска причин медленной работы системы, удаления вредоносного ПО и выявления конфликтующих драйверов или служб.
