# Практическая работа №8
## Тестирование персонального компьютера 
## с помощью утилиты Passmark Performance Test

### Цель работы
Провести финальное тестирование стабильности системы для последующего формирования технического задания по модернизации и внедрению компьютерной системы в учебный процесс.

### Теоретические сведения
**PerformanceTest** — это комплексное программное решение для всесторонней оценки производительности персонального компьютера. Программа включает 27 стандартизированных тестов, сгруппированных в 7 категорий: вычислительные возможности процессора, 2D- и 3D-графика, работа с дисковыми накопителями, оперативная память, оптические приводы и сценарии многозадачности. Утилита позволяет сопоставить производительность текущей системы с мировыми эталонными значениями.

| **Integer Math / CPU PhotoWorxx** | 14,900 MPix/s | 10,200 MOps/Sec |

**Анализ различий:**
- AIDA64 ориентирована на пиковые теоретические показатели с использованием оптимизированных инструкций (AVX, AES-NI).
- Performance Test моделирует более приближенные к реальности сценарии использования, поэтому значения выглядят скромнее.
- Оба инструмента сходятся в главном: производительность оперативной памяти находится на среднем уровне для платформы DDR3, а интегрированная графика является ограничивающим фактором.

### 3. Ответы на контрольные вопросы

**1. Для каких задач, на ваш взгляд, может применяться протестированная компьютерная система?**

Данная конфигурация (HP Compaq Pro 6300 SFF, Intel Core i5-2400, 8 ГБ RAM, SSD) оптимально подходит для:
- Ведения офисной документации и работы с электронными таблицами.
- Образовательной деятельности: изучение основ программирования, работа в офисных пакетах, подготовка презентаций.
- Комфортного серфинга в интернете, воспроизведения видео в формате Full HD.
- Функционирования в качестве терминала для подключения к удаленным рабочим столам.

**2. Какие компоненты системы, по вашему мнению, требуют первоочередного улучшения? Приведите конкретный пример.**

Наиболее слабым звеном является встроенный графический адаптер. Его апгрейд позволит:
- Расширить функциональность системы за счет использования современных мультимедийных приложений.
- Подключить дополнительные мониторы с высоким разрешением.
- Ускорить выполнение некоторых вычислительных операций с использованием GPU.

**Конкретный пример модернизации:** Установка дискретной видеокарты низкопрофильного форм-фактора (Low Profile), например **NVIDIA GeForce GT 1030** или **AMD Radeon RX 550**. Данные модели не требуют дополнительного питания и совместимы с компактным корпусом SFF. После замены оценка за видеоадаптер повысится до 6-7 баллов.
