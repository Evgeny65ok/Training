### 2. Создание и удаление файлов и каталогов (в `/tmp`)

```bash
cd /tmp

# Каталоги
mkdir newdir1
ls -d /tmp/newdir1
rmdir newdir1

mkdir -p newdir1/newdir2
ls -d /tmp/newdir1/newdir2
rmdir -p newdir1/newdir2

# Файлы
touch newfile1
ls -l /tmp/newfile1

> newfile2
ls -l /tmp/newfile2
rm newfile2
```

### 3. Копирование и перемещение файлов (в `/tmp`)

```bash
cd /tmp
cp /etc/fstab /tmp
ls -l /tmp/fstab
ls -l /etc/fstab   # время отличается

cp -p /etc/passwd /tmp
ls -l /tmp/passwd  # время совпадает

mv fstab fstab_new
ls /tmp/fstab_new  # файл перемещён
```
<img width="507" height="419" alt="image" src="https://github.com/user-attachments/assets/58476beb-b83a-4bb0-b9b3-2e362db8cf87" />

### 4. Копирование и перемещение каталогов (в `/tmp`)

```bash
mkdir -p dir_a/dir_b

cp -r dir_a dir_a_copy1
ls -ld /tmp/dir_a*

cp -ra dir_a dir_a_copy2
ls -ld /tmp/dir_a*  # атрибуты сохранены

mv dir_a dir_a_new
ls -ld dir_a_new
```

### 5. Шаблоны (glob) (в `/tmp`)

```bash
touch file1 file2 file3 file4
ls -l file*
rm file*
ls -l file*  # файлы удалены
```

### 6. Поиск файлов и каталогов

```bash
find ~ -name '.bashrc'
find ~ -name '*.png'
find ~ -type d -name 'Pictures'
```

### 7. Символические ссылки (в `/tmp`)

```bash
ln -s /etc/fstab fstab_link
ls -l fstab_link

ln -s /etc etc_link
ls -ld etc_link

# Битая ссылка
echo test > new_file
ln -s new_file new_file_link
rm new_file
ls -l new_file_link  # ссылка «мёртвая»
```

### 8. Оценка пространства

```bash
du -sh /etc
du -sh /usr
df -h
```
