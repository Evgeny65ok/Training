### 4. Отключение обновлений Windows 10

Выполнены следующие действия:
1. Запущен редактор локальной групповой политики (`gpedit.msc`)
2. Переход по пути: *Конфигурация компьютера → Административные шаблоны → Компоненты Windows → Центр обновления Windows*
3. В параметре «Настройка автоматического обновления» выбрано состояние **«Отключено»**
4. Изменения сохранены, редактор закрыт

**Результат:** В центре обновления Windows появилось сообщение о том, что некоторые параметры управляются организацией (обновления отключены).

### 7. Контрольные вопросы

#### 1. Что такое обновление?
Обновление — это исправление или дополнение к программному обеспечению, выпускаемое производителем для устранения ошибок, уязвимостей, улучшения функциональности или безопасности.

#### 2. Типы обновления Windows
- **Важные** — обеспечивают безопасность и стабильность, устанавливаются автоматически
- **Рекомендуемые** — улучшают работу, но не критичны
- **Необязательные** — драйверы, дополнительные программы, устанавливаются вручную
- **Обновления безопасности** — исправляют уязвимости
- **Критические обновления** — исправляют ошибки, не связанные с безопасностью
- **Пакеты обновлений (Service Packs)** — наборы всех предыдущих обновлений

#### 3. Центр обновления Windows
Центр обновления Windows — это встроенный компонент ОС, который автоматически или вручную проверяет, загружает и устанавливает обновления для Windows и других продуктов Microsoft.
