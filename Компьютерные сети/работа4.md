#### Шаг 4. Объединение файлов и редактирование
Все текстовые файлы из папки `Personal` были объединены в один общий файл `all.txt`. Затем в конец файла была добавлена строка с годом рождения.
```bash
cat Lab4/Info/Personal/Name.txt Lab4/Info/Personal/Date.txt Lab4/Info/Personal/School.txt > Lab4/Info/Personal/all.txt
echo "Год рождения: 2007" >> Lab4/Info/Personal/all.txt
cat Lab4/Info/Personal/all.txt
```
