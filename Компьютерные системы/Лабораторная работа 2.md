# Тестирование бумажного файлика.

| Тип тестирования             | Вопросы тестирования                                                                  |
|------------------------------|------------------------------------------------------------------------------------------------|
| **Black Box Testing**        | • Не зная конструкции, проверить: открывается/закрывается, держит бумаги, не ломается при обычном использовании |
| **Functional Testing**       | • Держатся ли бумаги внутри?<br>• Можно ли добавлять/удалять листы без повреждений? |
| **Usability Testing**        | • Удобно ли брать в руки?<br>• Легко ли открывать одной рукой?<br>• Читается ли этикетка под разным углом?<br>• Удобно ли носить в портфеле? |
| **Compatibility Testing**    | • Подходят ли листы А4, А5?<br>• Вмещается ли в стандартные полки/ящики?<br>• Совместим ли с ручками, маркерами для подписей? |
| **Performance Testing**      | • Сколько листов выдерживает без разрыва?<br>• Сколько циклов открывание/закрывание выдерживает?<br>• Какой максимальный вес бумаг держит? |
| **Component Testing**        | • Тестируем отдельно: металлический механизм, пластиковую обложку, кольца, этикетку<br>• Не отслаивается ли этикетка?<br>• Не гнутся ли кольца отдельно? |
| **Integration Testing**      | • После сборки: механизм + обложка + кольца — всё работает вместе?<br>• Не выпадают ли бумаги при переноске?<br>• Не заедает ли механизм при полной загрузке? |
| **Security Testing**         | • Нет ли острых краёв, о которые можно порезаться?<br>• Не содержит ли токсичных материалов?<br>• Не открывается ли самопроизвольно? |
| **Negative Testing**         | • Что будет, если вставить слишком много листов?<br>• Если попытаться закрыть наоборот?<br>• Если держать под углом 90 градусов с открытой застёжкой?<br>• Если положить тяжёлый предмет сверху? |
| **Stress Testing**           | • Нагрузить на 50% больше листов, чем заявлено<br>• Открывать/закрывать 500 раз подряд<br>• Резко дёргать за кольца<br>• Хранить под прессом |
| **UI Testing**               | • Чёткая ли разлиновка на этикетке?<br>• Удобно ли писать на поверхности этикетки?<br>• Не стираются ли надписи от трения?<br>• Эстетичный ли внешний вид? |
| **Entry Point Testing**      | • Как пользователь впервые взаимодействует: как понять, как открыть?<br>• Есть ли инструкция/интуитивный дизайн?<br>• Видны ли элементы управления (механизм открывания)? |
| **Documentation Testing**    | • Есть ли инструкция по использованию?<br>• Правильно ли указана вместимость?<br>• Указаны ли предупреждения (не гнуть, не перегружать)?<br>• Соответствует ли реальным возможностям? |
| **Automated Testing**        | • Роботизированное открывание/закрывание (циклы)<br>• Автоматическое измерение силы на разрыв колец<br>• Сканирование этикетки на читаемость |
| **Beta Testing**             | • Раздать офисным работникам для повседневного использования<br>• Собрать фидбэк: удобно ли, надёжно ли, какие улучшения предложить |
| **Verification**             | • Соответствует ли техническим требованиям? (размеры, материалы, вес)<br>• Проверка по ГОСТ/стандартам, если есть |
| **Validation**               | • Удовлетворяет ли потребности пользователя?<br>• Удобно ли для архивного хранения?<br>• Подходит ли для презентаций, ежедневного использования? |
